# zoom-api-attendance-taker
Google Apps Script to GET from Zoom API and write into a Google Sheet.

This repo is mostly to backup my code, but is public if anyone else needs this oddly specific usecase (I've scrubbed all personal IDs). Note that some extra custom filters exist that others may not need or want (MEETING_ID, ONLY_FOURTH_THURS, PARTICIPANT_BLACKLIST, etc), delete as needed.
